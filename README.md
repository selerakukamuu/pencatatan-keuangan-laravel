<h1 align="center">💰 Aplikasi Pencatatan Keuangan
</h1>

<p align="center">
Aplikasi sederhana berbasis Laravel untuk mencatat pemasukan dan pengeluaran seseorang.
</p>

<hr>

<h2>📌 Deskripsi</h2>
<p>
Aplikasi ini digunakan untuk mencatat transaksi keuangan mahasiswa, baik <b>pemasukan</b> maupun <b>pengeluaran</b>.
Data yang disimpan meliputi jenis transaksi, jumlah uang, dan keterangan.
</p>

<h2>✨ Fitur</h2>
<ul>
  <li>Menambahkan transaksi (Masuk / Keluar)</li>
  <li>Menampilkan daftar transaksi</li>
  <li>Edit data transaksi</li>
  <li>Hapus data transaksi</li>
  <li>Format otomatis ke Rupiah (Rp)</li>
</ul>

<h2>🛠️ Teknologi</h2>
<ul>
  <li>Laravel (Framework PHP)</li>
  <li>MySQL (Database)</li>
  <li>Bootstrap (UI)</li>
</ul>

<h2>🗄️ Struktur Database</h2>
<ul>
  <li><b>idtransaksi</b> (Primary Key, Auto Increment)</li>
  <li><b>jenistransaksi</b> (Varchar (10)) - (Masuk / Keluar)</li>
  <li><b>jumlah</b> (Integer)</li>
  <li><b>keterangan</b> (Varchar (50))</li>
</ul>
